Movie Recommendation Systems

Two Approaches

Collaborative Filtering

Collaborative filtering with cosine similarity is a recommendation technique that analyzes user-item interactions. By measuring the cosine similarity between users' preference vectors, it identifies similar users and recommends items based on their preferences.

Demo Link
Reduced SVD And Kmeans

Reduced SVD extracts latent features from user-item interaction data, reducing dimensionality while preserving essential user preferences. K-Means clustering then groups users based on these features, facilitating the identification of similar user segments for recommendations.
